# Análise de Crédito - Módulo 12
**Este repositório contém um projeto de análise de dados desenvolvido durante o Módulo 12 do curso de Ciência de Dados. O objetivo principal é identificar os principais impulsionadores para a concessão de altos limites de crédito**

### Objetivo:
**Realizar uma análise exploratória para responder: Quais fatores impactam diretamente o limite de crédito disponível para um indivíduo?.**

### O Dataset
**O conjunto de dados Ficticio Inclui informações como:**
* Nome,idade e Profissão
* Salário e limite de credito
* Histórico de inadimplência
* Estado Civil e posse de imóvel próprio

### Etapas do projeto
1. Criação do Dataframe
2. Escolha de 3 gráficos para identificar os fatores principais para os limites altos de crédito que foram:
   * Gráfico de Barras Horizontais
   * Gráfico de linha
   * Mapa de Calor
3. Justificativa da Escolha dos gráficos
